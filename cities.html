When resizing the browser, the row containing the city names (col-md) breaks first. This is because the city names are longer, and as the screen width decreases, there is not enough horizontal space to accommodate them, causing them to wrap to the next line. In contrast, the row containing the letters (col-sm-2) can adjust to smaller screen widths more flexibly, as each letter takes up less space.
